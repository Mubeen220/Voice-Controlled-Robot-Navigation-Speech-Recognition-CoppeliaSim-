# Voice-Controlled Pioneer P3-DX Robot Navigation

Overview
This project creates a real-time voice-controlled navigation interface for the Pioneer P3-DX robot in **CoppeliaSim**. It uses **SpeechRecognition** and **Google STT** to decode natural voice commands.

Features
- Accent-tolerant command recognition  
- Commands: forward, backward, left, right, stop  
- Speech-based human–robot interaction  
- ZMQ Remote API robot control  

Technologies
Python · SpeechRecognition · Google STT · CoppeliaSim · ZMQ API

Supported Commands
- “Forward”, “Ahead”, “Straight”  
- “Back”, “Reverse”  
- “Left”, “Turn left”  
- “Right”, “Turn right”  
- “Stop”, “Wait”
